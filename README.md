# Project Z - Backend

## Stack
 - NestJS
 - Mysql
 - PassportJS
 - Stripe

## Development
- Clone the repository
- Create .env
Create a file `.env` at the root and copy everything from `.env.sample` (available in root)
Make changes to the values
- Make sure mysql server is running
- Install dependencies 
```sh
npm i
```
- Start server
```sh
npm run start:dev
```

## Build
```sh
npm run build
```
